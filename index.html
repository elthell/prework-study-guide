HTML

  • HTML is formatted as: <'element attribute="value">content.
  • An element is a document component defined by a start tag, content, and an end tag.
  • An attribute modifies an element and is always placed in the opening tags.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • CSS is formatted as: selector { property: value; }.
  • A selector is an element/attribute that the CSS rule(s) is then applied to.
  • A property is a style or behavior assigned to the selector, specified with the value.
  • A margin inidicates how much space we want outside an element.
  • Padding indicates how much space we want around the content inside an element.

Git

  • All commands are proceeded by git.
  • git status: checks what branch we're on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add -A: stages the current changes
  • git commit -m "comment": commits changes with a comment
  • git pull origin main: makes sure the current branch is up to date before a pull request
  • git push orgigin branch-name: pushes changed to the origin/main branch

Javascript

  • Control flow is the order in which a computer executes code in a script
  • A variable is a named container that allows us to store data in our code
  • A for loop executes a block of code a number of times with different values
  • An if statement specified a block of code to be executed if condition(s) are met
  • An array is a container that allows us to store multiple values under one variable
  • A function is a block of code that can be reused whenever it's "called"
